💡 操作指导：
1. 输入四个测量点的 X、Y 坐标和 Z 高度值
2. 输入三个支点 A、B、C 的坐标（需构成三角形，测量点在内部精度最佳）
3. 点击「计算」获取 A、B 两点的建议调整量（C 点固定不动，正值=升高，负值=降低）
4. 调整后等待稳定，重新测量并再次输入，直至前后差/左右差 < 0.3 μm

📘 算法说明：支点 C 固定 (ΔC = 0)，采用重心坐标法 + 2×2 最小二乘求解 ΔA、ΔB，使四点高度方差最小。每测量点高度变化 = αᵢ·ΔA + βᵢ·ΔB，γ 权重虽存在但因 ΔC = 0 而不贡献。物理假设：刚性平台绕 C 点做倾斜调整。
